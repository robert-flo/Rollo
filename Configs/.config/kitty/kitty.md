# Kitty Keybindings

Kitty is a fast, feature-rich, GPU-based terminal emulator. [find out more](https://sw.kovidgoyal.net/kitty/)

| Keybinding | Action |
| ---------- | ------ |
| <kbd>Ctrl</kbd> + <kbd>-</kbd> | Zoom Out |
| <kbd>Ctrl</kbd> + <kbd>=</kbd> | Zoom In |
| <kbd>Ctrl</kbd> + <kbd>Shift + 1..8</kbd> | Select Tab 1..8 |
| <kbd>Ctrl</kbd> + <kbd>Tab</kbd> / <kbd>Ctrl</kbd> + <kbd>Shift + ]</kbd> | Next Tab |
| <kbd>Ctrl</kbd> + <kbd>Shift + Tab</kbd> / <kbd>Ctrl</kbd> + <kbd>Shift + [</kbd> | Previous Tab |
| <kbd>Ctrl</kbd> + <kbd>Shift + C</kbd> | Copy to Clipboard |
| <kbd>Ctrl</kbd> + <kbd>Shift + V</kbd> | Paste from Clipboard |
| <kbd>Ctrl</kbd> + <kbd>Shift + N</kbd> | New OS Window |
| <kbd>Ctrl</kbd> + <kbd>Shift + T</kbd> | New Tab |
| <kbd>Ctrl</kbd> + <kbd>Alt + Enter</kbd> | Split Horizontally (hsplit) |
| <kbd>Ctrl</kbd> + <kbd>Alt + S</kbd> | Split Vertically (vsplit) |
| <kbd>Ctrl</kbd> + <kbd>Alt + R</kbd> | Rotate Layout |
| <kbd>Ctrl</kbd> + <kbd>Alt + Left/Right/Up/Down</kbd> | Navigate splits (Arrows) |
| <kbd>Ctrl</kbd> + <kbd>Alt + H/J/K/L</kbd> | Navigate splits (Vim keys) |
| <kbd>F11</kbd> / <kbd>Super</kbd> + <kbd>Enter</kbd> | Toggle Fullscreen |
